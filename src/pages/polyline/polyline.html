Polyline Class

Create

map.addPolyline()
Add a polyline.

Methos

setPoints()
Change the polyline points.
getPoints()
Return an instance of the BaseArrayClass.
You can modify the points.
setGeodesic()
When true, edges of the polygon are interpreted as geodesic and will follow the curvature of the Earth.
getGeodesic()
Return true if the polyline is geodesic.
setVisible()
Change visibility of the polyline.
getVisible()
Return true if the polyline is visible.
setClickable()
Change clickablity of the polyline.
getClickable() Return true if the polyline is clickable.
setStrokeColor()
Change the polyline color.
getStrokeColor() Return the current polyline color.
setStrokeWidth()
Change the polyline stroke width.
getStrokeWidth() Return the current stroke width (unit: pixel).
setZIndex()
Change the polyline zIndex order.
getZIndex()
Return the current polyline zIndex.
remove()
Remove the polyline.
